Developers Home Commands

Commands

>help

Shows all the commands available.

>ping

Checks connectivity with discord's servers.

>say

Repeats whatever is said.

>covid

Check Covid Information from all countries.

>kick

kicks the mentioned member from the server.

>invite

gives the bot's invite link.

>clear

Clears messages.

>embedsay

Repeats a message but it will be embeded.

>ticket

Ticket System.

>close

Closes a ticket.

>setlogs

Sets a logs channel for the open/close tickets

And many more...
